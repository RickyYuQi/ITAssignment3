Skills and Jobs

Project manager

The Project manager would be in charge of leading and guiding groups as well as delegating tasks and keeping people organised and on schedule. They would break the project down into segments with deadlines for each individual. It would be necessary to have good leadership and communication skills in order to work efficiently with the group as well as direct them. Furthermore, being able to solve logistical oversights would be a skill highly valued in such a role in order to be able to tie together the project if something were to be found lacking down the road.

Professional use case

1. Lead software engineer

They would be in charge of writing code for the application, the tracking and working with the UI designer and the Database engineer to create the finished product. One of the more leading roles, they would be working with other members constantly in order to lay out their work in the code. Having good teamwork skills would be very beneficial as below the Project Manager they would be one of the most critical roles, working thoroughly with other members of the team.

2. Database engineer

A database engineer would manage the official scores of games, points and any other information collected, storing it by use of a database. They would need to work with the lead software engineer in order to implement the database into the app as well as connecting the tracking feature with the database for automatic inputs.

3. Software engineer

The secondary software engineer would work closely with the Lead software engineer on tasks that were delegated to them so that the Lead software engineer could work on more pressing matters. Having good time management skills as well as strong collaborative capabilities would be paramount in not slowing down the Lead software engineers work flow and ultimately keeping the project on track.

4. UI Designer

The UI Designer would be necessary in creating a well polished application, with user friendly design choices. This would allow for users to use the app efficiently, learn how to use it easily and find it easy to navigate. As well as enjoy the design could entice them to use the app more often. The UI designer would need good teamwork skills as they would work closely with the Lead software engineer who would implement their designs into the application.
